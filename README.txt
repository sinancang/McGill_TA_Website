McGill TA Website for COMP 307 Final Project


Folder Description
routes: contains php files that determine what html/php response to send when requests are made.
matter: php/html files that make up the DOM. php files are used for dynamic content.
utils: utility files that may be used be called by other php scripts. They are placed in separate files because they may be used by multiple scripts.
js: js files that are run on the front-end.
css: css styling for front-end.
db: "Database". csv files that are used for generating dynamic content.
